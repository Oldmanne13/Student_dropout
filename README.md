# Den stillede opgave:

I denne opgave skal der arbejdes med datasettet ”Student Dropout Prediction Dataset” fra Kaggle.com .

Supervised Learning:
Ved hjælp af supervised learning skal det undersøges hvilke af de faktorer familieindkomst, internet og daglige rutiner (transporttid og studievaner) der kan anvendes til at forudsige risikoen for frafald. Fokus ligger på at identificere tidlige advarselstegn, før de udmønter sig i dårlige akademiske resultater.
Vær åben for at arbejde med dataset for at optimere forudsigelsen, eksempelvis er det relevant at inkludere GPA (karakterer)?  

Unsupervised Learning:
Med unsupervised learning udforskes datasættet for at identificere skjulte mønstre og naturlige grupperinger af studerende. Formålet er at afdække, om der findes 'at-risk' profiler baseret på kombinationer af parametre, som ikke umiddelbart er logisk indlysende, men som har stor statistisk betydning for gennemførelsen.
